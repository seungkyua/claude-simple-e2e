# PRD: 장바구니 합계 계산기

## 개요

쇼핑몰의 장바구니에 담긴 상품들의 합계 금액을 계산하는 모듈.
세금, 할인, 배송비를 포함한 최종 결제 금액을 산출한다.

---

## 기능 요구사항

### 1. 상품 합계 계산
- 장바구니에 담긴 상품 금액의 합계를 계산한다
- 장바구니가 비어있으면 0을 반환한다
- 상품 수량 × 단가로 각 항목 금액을 계산한다

### 2. 세금 계산
- 부가세(VAT) 10%를 상품 합계에 적용한다
- 세금은 소수점 이하 반올림한다

### 3. 할인 적용
- 쿠폰 코드로 정률 할인(%)을 적용할 수 있다
- 할인은 세금 계산 전 상품 합계에 적용한다
- 유효하지 않은 쿠폰 코드는 에러를 반환한다
- 할인 후 금액이 0 미만이 될 수 없다

### 4. 배송비 계산
- 기본 배송비는 3,000원이다
- 상품 합계(할인 후)가 50,000원 이상이면 배송비 무료
- 배송비는 세금 계산에 포함하지 않는다

---

## 제약 조건

- 상품 가격은 0보다 커야 한다
- 수량은 1 이상의 정수여야 한다
- 쿠폰 할인율은 1~100 사이의 정수(%)여야 한다
