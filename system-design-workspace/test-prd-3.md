# PRD: 실시간 협업 할일 관리 (TaskFlow)

## 개요

소규모 팀(2~10명)을 위한 실시간 협업 할일 관리 웹앱.
칸반보드 형식으로 태스크를 관리하며, 팀원 간 실시간 동기화를 지원한다.

---

## 기능 요구사항

### 1. 태스크 CRUD
- 태스크 생성 (제목, 설명, 우선순위, 담당자, 마감일)
- 태스크 수정 및 삭제
- 드래그앤드롭으로 상태 변경 (To Do → In Progress → Done)
- 태스크에 라벨/태그 추가

### 2. 칸반보드
- 기본 3개 컬럼 (To Do, In Progress, Done)
- 커스텀 컬럼 추가/삭제/이름 변경
- 컬럼 내 태스크 순서 변경 (드래그앤드롭)
- 필터링 (담당자별, 라벨별, 우선순위별)

### 3. 실시간 동기화
- 다른 팀원의 변경 사항 즉시 반영
- 동시 편집 충돌 방지 (낙관적 잠금)
- 온라인/오프라인 상태 표시

### 4. 팀 관리
- 팀 생성 및 멤버 초대 (이메일)
- 역할 분리 (관리자/멤버)
- 팀별 보드 분리

### 5. 알림
- 태스크 할당 시 알림
- 마감일 임박 알림 (D-1, D-day)
- 댓글 알림

---

## 제약 조건

- 팀 최대 인원: 10명
- 보드당 최대 태스크: 500개
- 첨부파일: 이미지만, 최대 5MB
- 무료 서비스 (사내 도구이므로 과금 없음)
